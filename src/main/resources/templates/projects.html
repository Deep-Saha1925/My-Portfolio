Smart Student Housing

The Student Housing Management System is a web-based platform built using Java Spring Boot, Spring Security, Thymeleaf, and MySQL. It enables students to search and book rental properties, while owners can list and manage their properties. The system also includes admin features for managing users and maintaining platform integrity.

DATABASE MANAGEMENT USING FACE RECOGNITION

This project involves implementing a database management system integrated with face recognition technology. It allows for secure and efficient user authentication by leveraging facial biometrics. The system ensures data integrity and provides a seamless way to manage and access records using advanced recognition algorithms.

SCM(Smart Contact Manager)

Smart Contact Manager is a secure web application built using Java, Spring Boot, and MySQL. It allows users to register, log in, and manage their personal or business contacts. The app supports full CRUD operations, search functionality, and image uploads. Spring Security ensures role-based access control and user data protection.

Student Management System

The Student Management System project focuses on streamlining the management of student records, including enrollment, attendance, and performance tracking. It provides an intuitive interface for administrators, teachers, and students, ensuring efficient data handling and improved communication within educational institutions.
